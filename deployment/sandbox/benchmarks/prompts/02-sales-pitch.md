# 02 — Ten-slide sales pitch

## User message

> Create a 10-slide sales pitch for a B2B observability product called
> "Pulse". Differentiators: 50x faster query, open-source core, SSO for free.
> Three case studies: a fintech that cut MTTR by 60%, a retail chain that
> saved $1.2M on log storage, and a SaaS that replaced four tools with one.
> End with pricing: Starter $0, Team $299/mo, Enterprise contact sales. Use a
> coral/energetic palette.

## What "great" looks like

- Bold title slide with accent color on title.
- Problem / solution slides using split columns with a visual motif.
- "Why Pulse" slide using 3 icon-rows, each with a colored circle icon + header + one-line value prop.
- Case-study slides: 3 separate slides, each a stat-callout + a one-sentence
  narrative + customer name.
- A comparison table slide (Pulse vs status quo).
- A pricing slide with a 3-column table, middle column accent-highlighted.
- Closing slide with contact / next step CTA.
- Accent color is consistently "coral" (F96167 or equivalent), not blue.

## Common failure modes to watch for

- Defaults back to navy.
- Pricing rendered as three bullet points instead of a table.
- Case studies flattened into bullets on a single slide.
